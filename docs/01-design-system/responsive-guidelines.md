# Responsive Guidelines

## Breakpoints
| Token | Width |
|---|---:|
| `sm` | 640px |
| `md` | 768px |
| `lg` | 1024px |
| `xl` | 1280px |

## Mobile-first
Mọi section phải được thiết kế trước cho màn hình 360px. Sau đó mở rộng lên tablet và desktop.

## Navbar mobile
- Hiển thị tên rút gọn.
- CTA phone hoặc menu icon.
- Không nhồi nhiều link ngang.

## Hero mobile
- H1 tối đa 3-4 dòng.
- CTA stack dọc nếu không đủ ngang.
- Ảnh hero dưới text, ratio cố định.

## Services mobile
- Card 1 cột.
- Mỗi card có title, mô tả ngắn, optional icon.
- Tránh mô tả dài quá 3 dòng.

## Gallery mobile
- Grid 2 cột hoặc 1 ảnh lớn + 2 ảnh nhỏ.
- Ảnh phải có `object-fit: cover`.

## Checklist responsive
- [ ] Không tràn ngang ở 360px.
- [ ] CTA không bị xuống dòng xấu.
- [ ] Tên doanh nghiệp dài vẫn đọc được.
- [ ] Ảnh không méo.
- [ ] Section spacing không quá thưa trên mobile.
- [ ] Footer không lộ dữ liệu thiếu.
